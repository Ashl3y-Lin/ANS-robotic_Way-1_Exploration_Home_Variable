Variable

Variable it's a location in the memory of the EV3 Brick that can store a data value.

In other words, they are like magical boxes that hold different kinds of information, and you can use them to make your robot do all sorts of amazing things!

How to make a variable?

1- Press "Make a variable"

2- Name your "variable"

3- You can see that you have the variable you named.

You can make as many variables as you need.

Observations: Why is variable important for programming?

  • -> Remember Things: Just like we use notebooks to remember important information, variables help the robot remember things it needs to know.
  • -> Make Decisions: With variables, the robot can decide what to do based on what it "knows". For example, if it senses that it's too close to a wall, it can decide to turn away to avoid crashing.
  • -> Be Flexible: allow the robot to be flexible and adapt to different situations. If it needs to move faster, we can change the variable for speed.
  • If it needs to stop after a certain distance, we can adjust another variable.
  • -> Solve Problems: By using variables to store information and make decisions, the robot can solve problems on its own. For example, it can navigate through a maze, avoiding obstacles along the way.
